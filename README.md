# Sachithra's Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS. This portfolio showcases my work as a full-stack developer, content creator, and presenter.

## Features

- 🌓 Dark/Light mode support
- 🎥 Background video with dynamic overlay
- 📱 Fully responsive design
- ⚡ Fast and optimized performance
- 🎨 Modern UI with smooth animations
- 📝 Blog section
- 🖼️ Gallery showcase
- 🏆 Achievements timeline
- 📞 Contact form
- 🗺️ Interactive map

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- React Helmet
- Heroicons

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Project Structure

```
src/
├── components/     # Reusable components
├── contexts/       # React contexts
├── pages/         # Page components
├── assets/        # Static assets
│   ├── images/    # Image files
│   └── videos/    # Video files

```
